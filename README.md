Praticando Banco de Dados

Este repositório contém atividades e exercícios de SQL desenvolvidos em sala de aula.

## Estrutura
- `lista.sql`: arquivo com as queries da lista de exercícios
- Outros arquivos `.sql` poderão ser adicionados conforme novas atividades

## Objetivo
Praticar consultas SQL utilizando subconsultas, CTEs e outras técnicas de manipulação de dados.
